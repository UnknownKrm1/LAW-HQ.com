# Specification

## Summary
**Goal:** Build a single-page esports website for the LAW HQ clan with a black and red dark theme, aggressive visual style, and all clan content sections.

**Planned changes:**
- Apply global black/red dark theme using deep black backgrounds and crimson red accents, with Rajdhani or Barlow Condensed font from Google Fonts, geometric section dividers, scroll-reveal animations, and hover effects throughout
- Build a sticky navigation bar with LAW HQ wordmark, anchor links (Home, About, Leadership, Mission, Join, Discord), transparent-to-solid scroll behavior, and mobile hamburger menu
- Build a full-page Hero section with glowing "LAW HQ" title, tagline "Loyalty. Ambition. Winning.", animated entrance text, hero background image, and a "Join the Clan" CTA button scrolling to the Join section
- Build an "About Us" section with the provided clan description copy, red-accented heading, and scroll-reveal animation
- Build a "Leadership Team" section with two side-by-side cards (LAW EnergeT1cc — Founder, LAW KrmFv! — Presidential Board) each with placeholder avatar, red accent border, and provided descriptions; stacks on mobile
- Build an "Our Mission" section with the provided mission copy and a "What We Offer" section listing five feature items with red accent icons/markers, both with scroll-reveal
- Build a "Join LAW HQ" section with heading "We Are Recruiting", provided recruitment copy, and a red "Apply Now" button linking to https://discord.gg/WRadjbztnW (opens in new tab)
- Build a "Discord" section with a "Join Our Discord" button (links to https://discord.gg/WRadjbztnW, new tab), community description, three roles list (Members, Competitive Team, Staff), a static "350+ Members" badge, and placeholder social icon links for Twitter/X, YouTube, Instagram, TikTok (href="#")
- Build a footer with LAW HQ name, tagline, placeholder social icons, and copyright line "© 2025 LAW HQ. All rights reserved."

**User-visible outcome:** A fully styled, single-page LAW HQ esports website with all sections, navigation, animations, and Discord/social links ready to share.
